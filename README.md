# 🎬 STREAM VERSE

## 🚀 Tech Stack

- React + Vite + Tailwind
- Node.js + Express
- MongoDB

## 📦 Setup

### 1. Clone repo

```bash
git clone https://github.com/Prathamesh2801/Stream-Verse.git
```

### 2. Install dependencies

```bash
cd client && npm install
cd ../server && npm install
```

### 3. Setup environment

Create `.env` in server:

```
PORT=5000
MONGO_URI=your_mongo_uri
JWT_SECRET=your_secret
```

### 4. Run project

```bash
# server
npm run dev

# client
npm run dev
```
